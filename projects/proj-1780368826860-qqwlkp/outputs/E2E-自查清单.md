## 四、APK 打包步骤

### 前置条件
- HBuilderX 已登录 DCloud 账号
- `manifest.json` 中 AppID 已填写（`uni-app` 格式：`__UNI__XXXXXXX`）
- Android 权限已在 `manifest.json` 中声明（麦克风、存储）

### 打包步骤
```
1. HBuilderX 菜单 → 发行 → 原生App-云打包
2. 选择 Android
3. 证书选择：
   - 测试包：勾选「使用公共测试证书」
   - 正式包：上传自有 keystore（需提前生成）
4. 点击「打包」，等待约 5-10 分钟
5. 打包完成后，HBuilderX 下方控制台显示 APK 下载链接
6. 下载 APK → 传到 Android 手机 → 安装测试
```

### `manifest.json` 关键权限（确认已包含）
```json
{
  "app-plus": {
    "distribute": {
      "android": {
        "permissions": [
          "<uses-permission android:name=\"android.permission.RECORD_AUDIO\"/>",
          "<uses-permission android:name=\"android.permission.INTERNET\"/>",
          "<uses-permission android:name=\"android.permission.READ_EXTERNAL_STORAGE\"/>",
          "<uses-permission android:name=\"android.permission.WRITE_EXTERNAL_STORAGE\"/>"
        ]
      }
    }
  }
}
```

## 五、生产部署时 URL 切换

只需修改 `utils/config.ts` 第 12 行：

```typescript
// 改为：
const ENV: Env = 'prod'

// 并填写实际地址（第 23-24 行）：
apiBase: 'https://your-app.railway.app',
wsHost: 'your-app.railway.app',
```

修改后重新打包即可，**无需改动其他任何文件**。
